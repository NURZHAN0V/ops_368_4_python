# Работа на занятии 13‑11‑2025
## Инструкция по запуску программы  

### 1. Создание виртуального окружения  
Команда создаёт изолированную среду для проекта, чтобы избежать конфликтов зависимостей с другими Python‑проектами.  

```bash
python -m venv venv
```

**Результат:**  
В директории проекта появится папка `venv` со всеми необходимыми файлами окружения.

### 2. Разрешение выполнения скриптов в PowerShell (Windows)  
Эта команда требуется **только один раз** после создания окружения — она разрешает запуск скриптов в текущем пользовательском контексте.  

```bash
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

**Важно:**  
- Выполняется в PowerShell (не в CMD).  
- Система может запросить подтверждение — введите `Y` и нажмите Enter.

### 3. Активация виртуального окружения  
Активирует изолированную среду, чтобы все последующие команды `pip` и `python` использовали её зависимости.  

```bash
venv\Scripts\Activate.ps1
```

**Признаки успеха:**  
- В терминале появится префикс `(venv)` перед путём.  
- Все установки пакетов через `pip` будут применяться только к этому окружению.

### 4. Запуск приложения  
Запускает основной скрипт программы.  

```bash
python main.py
```

**Ожидаемое поведение:**  
Программа выведет меню взаимодействия и будет ждать ввода команды.

## MVP (Минимально жизнеспособный продукт)  

### 1. Инициализация списка клиентов  
Создан статический список клиентов в виде вложенных списков. Каждый элемент содержит: имя, телефон и Telegram‑никнейм.  

```python
client_list = [
    ["Олег", "+7 993 565 45 78", "@olegnastyle"],
    ["Дмитрий", "+7 993 657 45 78", "@dima"],
    ["Владимир", "+7 993 897 45 78", "@vova"],
]
```

**Примечание:**  
Данные хранятся в оперативной памяти — при перезапуске программы список сбрасывается к изначальному состоянию.

### 2. Вывод полного списка клиентов  
Простая демонстрация содержимого `client_list` через стандартную функцию `print`.  

```python
print(client_list)
```

**Результат:**  
Вывод в консоль всей структуры данных в формате списка списков.

### 3. Форматированный вывод имён клиентов  
Перебирает всех клиентов и выводит их имена с нумерацией. Используется переменная `count` для подсчёта позиций.  

```python
for client in client_list:
    print(f"{count}. {client[0]}")
    count = count + 1
```

**Особенности:**  
- `client[0]` — обращение к первому элементу вложенного списка (имя).  
- Нумерация начинается с значения `count`, которое должно быть инициализировано до цикла.

### 4. Добавление нового клиента  
Расширяет `client_list` новым элементом. Данные запрашиваются у пользователя через `input`.  

```python
client_list.append([
    input("Имя: "),
    input("Номер телефона: "),
    input("Телеграм: ")
])
```

**Логика:**  
- Создаётся новый список из трёх вводимых значений.  
- Метод `append()` добавляет его в конец `client_list`.

### 5. Интерактивное меню программы  
Бесконечный цикл `while True`, предоставляющий пользователю выбор действий:  
- Просмотр всех клиентов.  
- Добавление нового клиента.  
- Удаление клиента (функционал частично реализован).  

```python
while True:
    print(
        "1 - посмотреть всех килентов\n"
        "2 - добавить нового клиента\n"
        "3 - удалить клиента из списка\n"
    )
    value = input("Вы: ")
    
    # посмотреть всех клиентов
    if value == "1":
        print() # пустая строка
        count = 1
        for client in client_list:
            print(f"{count}. {client[0]}")
            count = count + 1
        print() # пустая строка

    # добавить нового клиента
    elif value == "2":
        print() # пустая строка
        client_list.append([
            input("Имя: "),
            input("Номер телефона: "),
            input("Телеграм: ")
        ])
        print() # пустая строка
```

**Детали реализации:**  
- Меню выводится повторно после каждого действия.  
- Для просмотра клиентов используется нумерация с обнулением `count = 1`.  
- При добавлении клиента входные данные сразу сохраняются в `client_list`.  
- Пункт «удалить клиента» заявлен, но код для него отсутствует.
