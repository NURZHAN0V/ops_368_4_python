**Задание. Проверка на дубликаты Telegram-никнеймов**

*Что нужно сделать:*  
Добавь проверку на дубликаты Telegram-никнеймов в функцию `client_add()`. Теперь программа должна проверять не только телефон, но и Telegram-никнейм — если клиент с таким же никнеймом уже существует, его нельзя добавить повторно.

*Требования:*  
1. После проверки телефона добавь ещё одну проверку для Telegram-никнейма (`client[3]`).  
2. Если найден клиент с таким же Telegram-никнеймом, выведи сообщение:  
   ```
   Клиент с Telegram @[никнейм] уже есть в базе
   ```  
3. Используй `return` для выхода из функции, если дубликат найден.  
4. Проверка должна происходить **до** добавления клиента в список.

*Пример кода:*  
```python
# После проверки телефона добавь:
for client in client_list:
    if client[3] == user_telegram:
        print(f"Клиент с Telegram {user_telegram} уже есть в базе")
        return
```

*Подсказки:*  
- Telegram-никнейм хранится в `client[3]` (четвёртый элемент списка).  
- Проверку можно добавить сразу после проверки телефона, используя тот же цикл `for` или отдельный.  
- Убедись, что обе проверки (телефон и Telegram) выполняются до `client_list.append()`.

*Цель:*  
Закрепить навыки работы с проверками на дубликаты и научиться проверять несколько полей одновременно для обеспечения уникальности данных.
