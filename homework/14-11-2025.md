### 5 заданий для закрепления материала

**Задание 1. Добавь поле «Email»**  
*Что нужно сделать:*  
- В список `client_list` добавь пятое поле — электронную почту для каждого клиента.  
- Обнови функцию `client_add()`, чтобы при выводе показывался и email.  
- При добавлении нового клиента (в `client_read()`) запроси у пользователя email и сохраните его в списке.  

*Цель:*  
Отработать работу с расширенной структурой данных и синхронизацию вывода/ввода.

**Задание 2. Проверяй ID перед удалением**  
*Что нужно сделать:*  
В функции `client_remove()` добавь проверку:  
- Если введённый ID **не существует** в списке, выведи сообщение:  
  ```
  Ошибка: клиента с ID [номер] не найдено!
  ```  
- Используй флаг‑переменную (например, `found = False`), чтобы отследить, был ли клиент найден.  

*Подсказка:*  
Проверяй существование ID **до** попытки удаления. Если клиент не найден — не пытайся удалять.

*Цель:*  
Научиться обрабатывать ошибки и избегать сбоев программы.

**Задание 3. Поиск клиента по имени**  
*Что нужно сделать:*  
Добавь в `clients.py` новую функцию `client_search()`:  
1. Запроси у пользователя имя для поиска.  
2. Пройди по списку клиентов и найди всех, чьё имя **содержит** введённую подстроку (используй `in`).  
3. Выведи найденные записи в формате:  
   ```
   Найден клиент:
   ID: 1
   Имя: Олег
   Телефон: +7 993...
   Телеграм: @olegnastyle
   ```  
4. Если клиентов не нашлось — выведи:  
   ```
   Клиенты с именем «...» не найдены.
   ```

*Цель:*  
Освоить поиск по строкам и работу с условными операторами.

---

**Задание 4. Сохрани данные в файл**  
*Что нужно сделать:*  
Добавь в программу сохранение списка клиентов в текстовый файл `clients.txt` при выходе:  
1. В конце цикла `while True` (в `main.py`) добавь проверку: если пользователь вводит `"save"`, программа сохраняет данные.  
2. В `clients.py` создай функцию `save_clients()`:  
   - Открывай файл в режиме записи (`"w"`).  
   - Для каждой записи в `client_list` записывай строку вида:  
     ```
     1;Олег;+7 993...;@olegnastyle;oleg@mail.com
     ```  
     (разделитель — точка с запятой).  
3. При запуске программы (в начале `main.py`) добавь загрузку данных из файла (если он существует). Для этого:  
   - Создай функцию `load_clients()` в `clients.py`.  
   - Читай файл построчно, разбивай строки по `";"` и добавляй в `client_list`.  

*Цель:*  
Познакомиться с базовыми операциями чтения/записи файлов.

**Задание 5. Улучши меню**  
*Что нужно сделать:*  
Обнови меню в `main.py`:  
1. Добавь новый пункт:  
   ```

   5 — сохранить данные
   ```  
2. Свяжи новые пункты с функциями `client_search()` и `save_clients()`.  
3. Добавь пункт `"exit"` для выхода из программы.  
4. Если пользователь вводит **нечисловой символ** или **неверный номер**, выведи:  
   ```

   ```  

*Дополнительно (по желанию):*  
Сделай так, чтобы после выполнения любого действия (кроме `exit`) меню появлялось снова.
