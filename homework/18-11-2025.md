**Задание 5. Улучши систему авторизации**  
*Что нужно сделать:*  
Добавь возможность повторного ввода логина и пароля при ошибке:  
1. В файле `auth.py` измени функцию `login()` так, чтобы она давала пользователю **3 попытки** для ввода правильных данных.  
2. Если все 3 попытки неудачны — выведи:  
   ```
   Доступ запрещён. Превышено количество попыток.
   ```  
   и верни `False`.  
3. Если введены правильные данные — сразу верни `True` и не запрашивай повторно.

*Пример структуры:*  
```python
def login():
    attempts = 3  # количество попыток
    while attempts > 0:
        login = input("Логин: ")
        password = input("Пароль: ")
        
        if login_admin == login and password_admin == password:
            return True
        else:
            attempts = attempts - 1
            print(f"Неверный логин или пароль. Осталось попыток: {attempts}")
    
    print("Доступ запрещён. Превышено количество попыток.")
    return False
```
