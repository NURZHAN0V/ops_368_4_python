**Задание 3. Проверка ввода при добавлении клиента**  
*Задача:*  
Добавьте проверку, что пользователь ввёл **непустые значения** при добавлении нового клиента. Если какое‑то поле оставлено пустым, выведите сообщение:  
```
Ошибка: все поля должны быть заполнены!
```  
и не добавляйте запись в список.


*Пример кода для проверки:*  
```python
name = input("Имя: ")
phone = input("Номер телефона: ")
telegram = input("Телеграм: ")
email = input("Email: ")

if name and phone and telegram and email:
    client_list.append([name, phone, telegram, email])
else:
    print("Ошибка: все поля должны быть заполнены!")
```

**Задание 5. Сохранение данных в файл**  
*Задача:*  
Добавьте в программу возможность сохранять список клиентов в текстовый файл `clients.txt` при выходе.  

*Шаги:*  
1. В конце цикла `while True` добавьте проверку: если пользователь вводит `"exit"`, программа сохраняет данные и завершается.  
2. Для сохранения используйте:  
   ```python
   with open("clients.txt", "w", encoding="utf-8") as file:
       for client in client_list:
           file.write(" | ".join(client) + "\n")
   ```
3. При запуске программы (до цикла `while`) добавьте загрузку данных из файла (если он существует):  
   ```python
   if os.path.exists("clients.txt"):
       with open("clients.txt", "r", encoding="utf-8") as file:
           for line in file:
               client_list.append(line.strip().split(" | "))
   ```

*Цель:*  
Познакомиться с базовыми операциями чтения/записи файлов.
